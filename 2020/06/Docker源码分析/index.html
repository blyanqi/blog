Moby:Moby是由Docker创建的一个开源框架，用于组装专门的容器系统.

概念

  • Docker Client（客户端）
  • Docker Daemon (服务)
    • Docker Server
      • http server
      • router
      • Handler
    • Docker Engine
      • Job
  • Docker Driver（驱动）
    • graph Driver
      • graph DB
      • Repository
    • network Driver
      • bridge
      • IP
      • PORT
      • NETWORK interface
        • netlink
    • exec Driver
      • native
        • namespace
        • cgroups
  • Docker Container
  • Docker Registry
