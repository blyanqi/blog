源文件声明规则

一个源文件中只能有一个public类
一个源文件可以有多个非public类
源文件的名称应该和public类的类名保持一致。
package语句应该在源文件的首行。
import语句放在package语句和类定义之间。
import语句和package语句对源文件中定义的所有类都有效。

数据类型

  • 内置数据类型
    Java语言提供了八种基本类型。六种数字类型（四个整数型，两个浮点型），一种字符类型，还有一种布尔型。
    byte 8、short 16、int 32、long 64
    float 32、double 64
    boolean true|false
    char 16 unicode
  • 引用数据类型
    引用类型指向一个对象，指向对象的变量是引用变量。
    对象、数组都是引用数据类型。
    所有引用类型的默认值都是null。

自动类型转换

byte,short,char—> int —> long—> float —> double
在把容量大的类型转换为容量小的类型时必须使用强制类型转换。
强转导致溢出或损失精度
浮点数到整数的转换是通过舍弃小数得到，而不是四舍五入

访问控制修饰符 (实际使用中常用 private、public)

  • default 同一包内可见 package
  • protected 同一包内的类和所有子类可见
  • private 同一类内可见
  • public 对所有类可见

非访问修饰符

static 修饰符，用来修饰类方法和类变量。
final 修饰符，用来修饰类、方法和变量

final 修饰的类不能够被继承
修饰的方法不能被继承类重新定义
修饰的变量为常量，是不可修改的

abstract 修饰符，用来创建抽象类和抽象方法。
synchronized 和 volatile 修饰符，主要用于线程的编程。
synchronized 关键字声明的方法同一时间只能被一个线程访问。
