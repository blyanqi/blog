注册一个自己的域名

freenom
去 freenom 注册一个自己的域名
网站有点卡，可以尝试多试几次。
management tools -> nameservers -> Use custom nameservers (enter below)->
把 dnspod 的 ns 地址填进去即可

使用 dnspod 进行域名解析

dns pod
控制台->添加域名->输入新注册的域名->点入域名-> 把ns地址粘到 freenom 中。
-> 点击刷新 -> 一会就生效

在 GitHub 中配置域名

github page 添加自定义域名的方法
repo->setting->GitHub Pages ->Custom domain-> blog.yanqi.cf（我申请的域名）
保存会在repo 里生成一个 CNAME 文件
需要把这个文件写到 hexo 主题下，否则每次发布都会覆盖库的文件
echo “blog.yanqi.cf” >themes/next/source/CNAME

添加 CNAME类型 DNS 解析

blog CNAME blyanqi.github.io

修改 hexo 配置

url: http://blog.yanqi.cf/
root: /

为 blog 添加统计分析

使用百度统计

添加网站-> 获取代码
blog.yanqi.cf
https://blog.yanqi.cf/index.html

获取key,把这一行？后的key 添加到主题配置文件
hm.src = “https://hm.baidu.com/hm.js?“;

修改主题配置文件 themes/next/_config.yml
baidu_analytics: ‘829697829f3xxxxxxx’

修改百度分析文件
themes/next/layout/_third-party/analytics/baidu-analytics.njk
直接把代码复制到这个文件，否则直接加key。百度统计检查不出来。
